# Vue Project (with Vuetify)

This is a Vite + Vue 3 starter project with Vuetify added and a simple `HomePage` component.

To run locally:

```powershell
cd "c:\Users\Admin\Desktop\Projekts\Frontend\vue-project"
npm install
npm run dev
```

The `HomePage` component is `src/components/HomePage.vue` and uses Vuetify components. The frontend can consume the Laravel backend API at `http://127.0.0.1:8000/api/home`.

### Scripts

```sh
npm run dev    # starts vite dev server
npm run build  # build for production
npm run lint   # lint project
```
